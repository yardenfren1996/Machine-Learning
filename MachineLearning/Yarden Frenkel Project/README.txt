Welcome to Computer Vision Final Project by Yarden Frenkel

To get the results file do the following staff:

1. make sure -  the Model 'model.h5' file, the DataBase 'SynthText_test.h5' and the Main script 'main.py'
    are on the same containing folder
    otherwise go to the 'main.py' file and change the paths.
2. open terminal in the containing folder
3. install the requirements with the command "pip install -r requirements.txt"
4. run the command "python main.py"
5. follow the instructions and wait for the results
